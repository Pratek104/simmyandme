IMPORTANT:
You need to add your own files to this folder for the website to work perfectly!

1. Music:
   - Name your music file: music.mp3
   - Format: MP3

2. Photos:
   - You need 10 photos for the memory lane.
   - Name them exactly:
     photo1.jpg
     photo2.jpg
     photo3.jpg
     ...
     photo10.jpg

   (Supported formats: .jpg, .png, .jpeg - just rename them or update index.html if you use png)
